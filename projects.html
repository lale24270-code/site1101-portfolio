Todo List Application

A dynamic todo list application built with vanilla JavaScript. Users can add, edit, delete, and mark tasks as complete. The app uses local storage to persist data between sessions.

Technologies: JavaScript, HTML, CSS, Local Storage

Key Features:

  • Add and delete tasks
  • Mark tasks as complete
  • Persistent data storage
  • Clean and intuitive UI

Weather Application

A weather application that fetches real-time weather data from a public API. Users can search for any city and view detailed weather information including temperature, humidity, and wind speed.

Technologies: JavaScript, REST API, HTML, CSS

Key Features:

  • Search weather by city name
  • Real-time weather data
  • Display temperature and conditions
  • Responsive weather cards
