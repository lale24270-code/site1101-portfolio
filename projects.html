Building logic gates

This academic project focuses on understanding and building basic logic gates such as AND, OR, and NOT. The project helped me learn fundamental concepts of digital logic and logical thinking, which are essential for computer science and information technologies.

Kit box:

  • Solderless breadboard - 1 piece
  • Power supply module - 1 piece
  • Resistor (330 Ohm) - 1 piece
  • Small pushbutton - 2 pieces
  • Resistor (2,000 Ohm) - 3 pieces
  • NPN transistor (PN2222) - 3 pieces
  • Light-emitting diode (LED) - 3 pieces
  • Jumper wire - 11 pieces

Todo List Application

A dynamic todo list application built with vanilla JavaScript. Users can add, edit, delete, and mark tasks as complete. The app uses local storage to persist data between sessions.

Technologies: JavaScript, HTML, CSS, Local Storage

Key Features:

  • Add and delete tasks
  • Mark tasks as complete
  • Persistent data storage
  • Clean and intuitive UI
